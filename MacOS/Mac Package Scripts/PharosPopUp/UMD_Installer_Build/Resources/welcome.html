UMD Library Printers Installer

This installer will set up all UMD library printers on your Mac.

What will be installed:

  • Pharos Popup printing client
  • All UMD library printers (17 printers total)
  • Required printer drivers

Libraries included:

  • McKeldin Library
  • Architecture Library
  • Art Library
  • Engineering & Physical Sciences Library (EPSL)
  • Hornbake Library
  • Performing Arts Library (PAL)
  • Maryland Room

Note: Administrator password required for installation.
